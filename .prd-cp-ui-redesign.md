# UI 重新设计：配色和位置

## 功能描述

修复聊天面板位置，重新设计导航提示以配合页面灰色调。

## 成功标准

1. 聊天面板固定在右下角，尺寸不变
2. 导航提示配合 Core 的 slate 灰色调
3. 更高级的科幻效果

## 实现方案

### 1. 聊天面板
- 位置: fixed bottom-20 right-6
- 尺寸: w-80 h-[480px] (绝对固定)
- overflow 控制消息区域滚动

### 2. 导航提示
- 使用 slate 灰色调而非紫色
- 右下角滑入的玻璃卡片
- 扫描线 + 边框发光效果

- [ ] 聊天面板尺寸固定
