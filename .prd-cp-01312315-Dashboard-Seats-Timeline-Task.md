## PRD - Dashboard 实时更新：Seats + Timeline + Task Chain 自动刷新

**需求来源**: Brain 自动调度任务。Dashboard 页面需要实时显示 Cecelia 执行状态，当前依赖手动刷新或固定 30s 轮询，体验不佳。

**功能描述**:
为 Core Dashboard 添加 SSE (Server-Sent Events) 实时推送机制，覆盖三个核心模块：
1. **Seats（并发槽位）**: 实时显示每个 Cecelia 执行槽的状态变化（空闲/运行中/完成）
2. **Timeline（任务时间线）**: 实时推送任务进度更新（checkpoint 状态变更）
3. **Task Chain（任务链）**: 实时显示当前执行中的任务步骤及其子步骤

**技术方案**:
- 后端：Express 路由添加 SSE endpoint (`/api/cecelia/stream`)，周期性推送 seats/overview 状态
- 前端：创建 `useSSE` hook，组件订阅 SSE 事件自动更新 UI
- 降级：SSE 连接失败时回退到 30s 轮询

**涉及文件**:
- `apps/core/src/dashboard/routes.ts` - 添加 SSE endpoint
- `apps/core/features/dashboard/pages/DashboardHome.tsx` - 接入 SSE 数据
- `apps/core/features/dashboard/hooks/useSSE.ts` - 新建 SSE hook
- `apps/core/features/dashboard/components/` - Seats/Timeline/TaskChain 组件（新建或修改）

**成功标准**:
- [ ] SSE endpoint `/api/cecelia/stream` 返回 `text/event-stream`，每 5s 推送 seats + overview 数据
- [ ] 前端 `useSSE` hook 自动连接 SSE，断连时回退到 30s 轮询
- [ ] Seats 组件实时显示槽位状态（占用/空闲/运行时长）
- [ ] Timeline 组件实时显示最近任务的 checkpoint 进度
- [ ] Task Chain 组件实时显示当前运行中任务的步骤详情
- [ ] SSE 连接断开后 3s 内自动重连

**非目标**:
- 不重构现有 Dashboard 页面布局
- 不引入 WebSocket（SSE 足够满足单向推送需求）
- 不修改 Autopilot Dashboard（`apps/dashboard/frontend/`），只改 Core Dashboard
