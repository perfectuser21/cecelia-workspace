## 验收标准

### 功能验收

- [ ] Block 创建：点击 "+" 按钮可选择并创建 5 种类型 block（text/heading/list/code/callout）
      Test: manual:在页面点击+按钮验证5种类型可创建
- [ ] Block 编辑：点击 block 可编辑内容，blur 时自动保存到 API
      Test: manual:编辑内容后blur，刷新页面验证数据保存
- [ ] Block 删除：点击删除按钮移除 block，API 同步删除
      Test: manual:删除block后刷新页面验证已删除
- [ ] 拖拽排序：拖动 handle 可调整顺序，松开时 API 同步 reorder
      Test: manual:拖拽调整顺序后刷新页面验证顺序保持
- [ ] API 同步：所有操作实时同步到后端
      Test: manual:通过curl验证API数据与页面一致
- [ ] 集成验证：在 Goal 详情页可展示并编辑 blocks
      Test: manual:打开Goal详情页验证BlockEditor渲染
- [ ] 响应式布局：移动端（< 768px）布局正常可操作
      Test: manual:Chrome DevTools切换移动端视图验证

### 测试验收

      Test: npm run build
