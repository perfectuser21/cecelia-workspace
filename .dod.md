# DoD - 提交 platform-data 删除操作

## 验收标准

### 功能验收

- [ ] `git status` 显示干净（无 deleted 文件）
      Test: manual:运行 git status 确认无 deleted 文件

- [ ] `apps/features/platform-data/` 目录不存在
      Test: manual:运行 ls apps/features/ 确认目录不存在

- [ ] 删除操作已提交到 git
      Test: manual:git log 显示删除 commit

### 质量验收

- [ ] Commit message 符合规范（refactor: 删除 platform-data）
      Test: manual:git log 查看 commit message

- [ ] PR 已创建并指向 develop
      Test: manual:gh pr view 显示 PR 信息

### CI 验收

- [ ] CI 检查通过
      Test: manual:gh run list 显示 success

- [ ] PR 已合并到 develop
      Test: manual:gh pr view 显示 Merged 状态
