# Template Changelog

## 2025-12-25

### Added
- **github-api**: GitHub API 调用模板
  - 调用 GitHub API 获取仓库信息（stars、repo 等）
  - 包含数据处理节点
  - 关键词: github, api, star, repo, 仓库, stars

- **code-node**: 代码处理节点模板
  - 使用 JavaScript 处理数据的通用模板
  - Webhook → Code → Respond 流程
  - 关键词: code, 逻辑, 处理, 计算, 脚本, js, javascript

- **data-compare**: 数据对比模板
  - 对比当前和历史数据的变化
  - 自动计算差值和增长趋势
  - 关键词: 对比, 比较, compare, diff, 增量, 变化

### Updated
- `index.json`: 添加 3 个新模板的索引信息
