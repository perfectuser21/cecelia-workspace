# VPS Health Check Monitor

VPS 健康监测系统，通过 n8n 自动化工作流定期检查服务器磁盘使用情况，当超过阈值时发送飞书告警通知。

## 功能概述

- **定时检查**：每小时自动触发一次 VPS 磁盘空间检查
- **远程执行**：通过 SSH 连接到 VPS 执行磁盘检查命令
- **智能告警**：磁盘使用率超过 80% 时触发飞书通知
- **异常处理**：捕获执行过程中的异常并发送告警

## 系统要求

- n8n 实例已部署并正常运行
- SSH 凭证配置完成（VPS: 146.190.52.84）
- 飞书 Webhook 配置完成
- VPS 支持 `df -h` 命令

## 工作流架构

```
Schedule Trigger (Cron: 0 * * * *)
    ↓
SSH 执行磁盘检查
    ↓
Code 解析磁盘数据
    ↓
IF 判断使用率 > 80%
    ├─ 是 → Feishu 发送告警
    └─ 否 → 流程结束

Error Trigger (异常捕获)
    ↓
Feishu 发送错误告警
```

## 告警信息内容

告警消息包含以下信息：
- 主机名
- 分区名称
- 总容量
- 已用容量
- 可用容量
- 使用率百分比

## 监控覆盖范围

- `/` 根分区
- `/home` 用户数据分区
- 其他已挂载分区
