# PRD: 修复 /cecelia 路由冲突 + 添加 CeceliaPage

## 需求来源
/cecelia 路由被 execution feature（位置5）注册为 CeceliaOverview，
导致 cecelia feature（位置9）的 CeceliaPage 永远无法匹配。

## 功能描述
- 从 execution/index.ts 移除 { path: '/cecelia', component: 'CeceliaOverview' } 顶级路由
- 在 features/index.ts 添加 cecelia feature 注册
- 添加 CeceliaPage（意识感重设计）和 CeceliaConfigPage

## 成功标准
- 访问 /cecelia 显示新 CeceliaPage（主动对话 + 警觉状态），不显示 CeceliaOverview
- npm run build 无报错
