# 导航科技感提示条

## 功能描述

为语音导航功能添加更有科技感的视觉反馈 - 一个从顶部滑入的浮动提示条。

## 成功标准

1. 导航时显示毛玻璃效果的提示条
2. 带有紫色发光动画
3. 扫描线动画增强科技感
4. 2秒后自动消失

## 实现方案

### NavigationToast 组件
- 固定在屏幕顶部居中
- 毛玻璃背景 (backdrop-blur)
- 紫色渐变发光效果
- 扫描线动画

### 触发时机
- executeFrontendTool('navigate_to') 时
- 自动 2 秒后消失

## 验收清单

- [ ] 导航时顶部出现提示条
- [ ] 毛玻璃效果正常
- [ ] 发光动画正常
- [ ] 2秒后自动消失
