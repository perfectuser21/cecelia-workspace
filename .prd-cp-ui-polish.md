# UI 优化：导航提示位置和语音别名

## 功能描述

优化导航提示条位置和添加更多语音命令别名。

## 成功标准

1. 导航提示条移到右下角（Cecelia 按钮上方）
2. 提示条使用进度条动画代替扫描线
3. 添加 "Command Center"、"OpCenter" 等导航别名
4. 语音说 "command center" 能正确导航到 orchestrator

## 变更内容

### 1. 导航提示条位置
- 从顶部居中 → 右下角
- 滑入方向从上往下 → 从右往左
- 动画改为底部进度条

### 2. 新增路由别名
- command center, commandcenter, command
- opcenter, op center
- control center, controlcenter
- 指挥中心, 控制中心
- task (单数)
- dashboard, 仪表盘
