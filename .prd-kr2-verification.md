# PRD: KR2 Verification - Nightly Loop 验收

## 背景

Phase 5 完成了 Memory + Planning + Commit + Nightly。现在需要验证 KR2：N8N 调度可自动触发 3 个任务。

## 功能描述

创建 KR2 验收脚本，验证 Nightly Loop 全链路：
Plan → Commit → Execute → Evidence → Memory

## 成功标准

- nightly 生成 plan（有 planId）
- commit 生成 tasks（至少 3 个）
- memory 记录 nightly_plan 事件
- 系统健康（DLQ=0, 无 degrade, 无 violations）

## 交付物

### scripts/verify-nightly-loop.sh

验证流程：
1. POST /api/system/plan/nightly 触发
2. 检查 plan 结构（tasks 有 why/expected_evidence）
3. 检查 committed tasks 数量
4. 检查 memory 中有 nightly_plan 事件
5. 检查系统健康（DLQ, Degrade, Assertions）
6. 输出 Summary 表格 + PASS/FAIL
