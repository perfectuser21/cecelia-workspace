# PRD: 移除 TaskDatabase 顶部表格的灰色边框

## 背景

用户反馈 TaskDatabase 页面（/work/tasks）顶部表格有个灰色边框，这个边框没有实际作用，影响视觉美观。

## 问题描述

在 TaskDatabase 页面，表格容器使用了 `border border-slate-700` 样式，在顶部和四周形成了一圈灰色边框。这个边框：
- 在视觉上将表格与背景分离，但没有必要
- 与 Notion 风格的简洁设计不符
- 用户认为是多余的装饰

## 解决方案

移除 `TaskTableView` 组件中表格容器的边框样式。

### 修改文件

`apps/core/features/planning/pages/TaskDatabase.tsx` 第 713 行：

```tsx
// 修改前
<div className="bg-slate-800 rounded-xl border border-slate-700 overflow-hidden">

// 修改后
<div className="bg-slate-800 rounded-xl overflow-hidden">
```

## 影响范围

- 只影响视觉样式
- 不影响功能
- 不影响布局和交互

## 验收标准

1. TaskDatabase 表格容器没有灰色边框
2. 其他样式（圆角、背景色）保持不变
3. 页面正常显示，无布局问题
