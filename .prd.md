# PRD: 合并 Staff + Skills 为单一 Team 页面

## 需求
- Staff（员工）和 Skills（技能库）是同一个系统的两部分，没必要分两个 tab
- 合并为单一 /system/team 页面，消除冗余导航

## 功能描述
- 上方：Area → Department → Worker 三层层级（含 model badge 和 skill 标签）
- 下方：Skills Registry（Skills / Agents 切换）
- 移除 /system/skills-registry tab，将 /system/staff tab 改为 /system/team
- 原路径加 redirect

## 涉及文件
- apps/core/features/system/pages/TeamPage.tsx（新建，合并逻辑）
- apps/core/features/system-hub/pages/SystemTabbed.tsx（改 tab）
