# PRD: Phase 4.5 - Organ Integrity Verification (器官一致性验证)

## 功能描述

创建 `verify-organs.sh` 脚本，检测系统是否存在多脑/多手/混层问题：

### Layer 1: 运行时单入口检测
- 端口暴露检查：只有 workspace:5212 对外开放
- brain/quality/n8n 只监听内网

### Layer 2: 接口级职责越界检测
- 调用 /api/system/assertions/validate
- 检测 boundary violations

### Layer 3: 代码结构级扫描
- 多脑扫描：orchestrator/queue/scheduler/dispatch 只在指定位置
- 多手扫描：skills/worker/execute 只在 workflows

## 成功标准

- [ ] verify-organs.sh 脚本
- [ ] Layer 1: 端口检测
- [ ] Layer 2: Assertions 调用
- [ ] Layer 3: 静态代码扫描
- [ ] 输出 BRAINS_FOUND / HANDS_FOUND / violations
