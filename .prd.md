## PRD - Python Semantic API v1

**需求来源**: Cecelia 双脑架构统一阶段2，Node.js Brain (5221) 需要调用 Python 语义感知层 (5220) 的标准 API
**功能描述**: 在 cecelia-semantic-brain/src/api/ 下创建 semantic_routes.py，提供四个语义 API 端点供 Node.js Brain 调用
**涉及文件**:
- `cecelia-semantic-brain/src/api/semantic_routes.py`（新建）
- `cecelia-semantic-brain/src/main.py`（修改，include router）

**成功标准**:
- [ ] POST /v1/semantic/embed 端点正常响应，调用 embedder 进行文本向量化
- [ ] POST /v1/semantic/search 端点正常响应，封装现有 fusion 搜索逻辑
- [ ] POST /v1/semantic/rerank 端点正常响应，对结果进行重排序
- [ ] GET /v1/semantic/health 端点返回 embedder 和 store 的状态信息
- [ ] 四个端点都有基本的错误处理和超时机制

**非目标**: 不修改现有 embedder/search_engine 实现、不改数据库 schema、不加认证鉴权
