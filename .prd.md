# Brain Decision Pack v2.0

## 目标
增强 Brain API 的 decision pack，提升 LLM 自省能力

## 功能
1. 修复 recent_decisions 的 action 名称（unknown -> 实际动作）
2. 添加 pack_version / generated_at / ttl_seconds
3. 添加 decision_mode（interactive/scheduled/incident）
4. 添加 action_constraints（幂等/安全闸门）
5. 增强 task_digest（带 id/title/status/updated_at/due_at）
6. 增强 system_health（可量化指标）

## 验收标准
- [x] action 不再显示 unknown
- [x] 包含元数据字段
- [x] 支持 mode 参数
