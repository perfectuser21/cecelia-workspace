# PRD: 将 Feature Dashboard 加入侧边栏菜单

## 问题
Feature Dashboard 页面已经存在，但只是一个隐藏路由，不在侧边栏菜单里。用户登录后看不到入口，必须手动输入 URL `/features` 才能访问。

## 解决方案
在导航配置中，将 Feature Dashboard 添加到侧边栏菜单：
- 位置：放在"账号管理"后面
- 图标：LayoutGrid
- 标签：Feature 总览

## 验收标准
1. 登录后在侧边栏能看到 "Feature 总览" 菜单项
2. 点击后能正常打开 Feature Dashboard 页面
3. 菜单图标和文字显示正确
