# PRD: Phase 4.1 - Trace ID 贯穿闭环

## 功能描述

实现 trace_id 贯穿整个闭环链路，让闭环失败时可快速定位哪个环节出问题。

## 成功标准

- [ ] trace_id 格式：`trc_YYYYMMDD_HHMMSS_<rand>`
- [ ] verify-closed-loop.sh 生成并输出 trace_id
- [ ] trace_id 通过 task metadata 传递
- [ ] 闭环验证输出能串联 task_id + trace_id
