# PRD: Orchestrator 详情页

## 目标

创建 Orchestrator 详情页，展示系统状态并支持对话式任务拆解。

## 功能

### 1. 状态面板（左侧）
- 今日焦点卡片
- Projects 树形结构
- OKR 树（Objective → Key Results）
- 当前任务列表（按优先级）

### 2. 聊天界面（右侧）
- 消息历史
- 输入框 + 发送按钮
- 操作按钮（创建任务等）

### 3. 联动效果
- 对话中提到的对象，左侧自动展开/高亮
- 回复中的引用可点击
- 操作执行后状态面板实时更新

## 范围

- `apps/core/src/orchestrator/routes.ts` - 聊天 API
- `apps/core/features/orchestrator/` - 前端页面
- `apps/core/features/index.ts` - 注册 feature

## 成功标准

- [ ] `/orchestrator` 页面可访问
- [ ] 状态面板显示 Projects/OKR/Tasks
- [ ] 聊天界面可发送消息并收到回复
- [ ] 聊天和状态面板联动
