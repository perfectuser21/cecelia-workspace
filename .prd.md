## PRD - localhost/Tailscale 自动登录

**需求来源**: 用户反馈飞书 OAuth 登录对个人使用场景过于复杂，希望通过 Tailscale 私有网络直接访问无需登录

**功能描述**: 当 hostname 是 localhost、127.0.0.1 或 100.x.x.x（Tailscale 网段）时，自动以超级管理员身份登录，跳过飞书 OAuth。外部域名访问保持不变。

**涉及文件**:
- `apps/dashboard/src/contexts/AuthContext.tsx` - 检测 hostname 自动注入 user/token

**成功标准**:
1. localhost 访问时自动登录为超级管理员，不显示登录页
2. 127.0.0.1 访问时同上
3. 100.x.x.x（Tailscale IP）访问时同上
4. 外部域名（*.zenjoymedia.media）访问保持飞书 OAuth 不变
5. 前端 vite build 通过

**非目标**:
- 不改后端 API
- 不改路由结构
- 不删除飞书 OAuth 代码（保留给外部域名使用）
