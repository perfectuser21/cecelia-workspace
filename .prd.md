# PRD: Cleanup Admin Routes and Add Accounts Feature

## 功能描述
1. 添加 `accounts` feature flag 让账号管理在侧边栏显示
2. 删除 Autopilot 中引用 Core 功能的管理员路由

## 背景
- 管理员页面 (ClaudeMonitor, VpsMonitor, N8n 等) 已迁移到 Core
- Autopilot 不应该引用这些路由，否则会报组件找不到
- 账号管理功能已添加导航但 feature flag 缺失

## 成功标准
1. `accounts` 在侧边栏显示
2. 无 "Page component not found" 警告
3. npm run build 通过
