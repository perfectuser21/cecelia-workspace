# PRD: Orchestrator UI 在 Brain Dashboard

## 背景

Orchestrator API 已通过 proxy 暴露到 Core (5212)，需要在前端显示 Orchestrator 状态。

## 目标

在 BrainDashboard 页面添加 Orchestrator 状态卡片，显示：
- 健康状态
- 索引的 chunks 数量
- Layer 2 状态（如有）

## 范围

- `apps/core/features/brain/pages/BrainDashboard.tsx`

## 验收标准

1. Brain Dashboard 页面显示 Orchestrator 状态卡片
2. 卡片显示 health status 和 indexed_chunks
3. 自动刷新与其他卡片同步
