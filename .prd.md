# PRD: Fix Core Instance Detection

## 功能描述
修复 `isCoreInstance()` 函数，使其能够正确识别 localhost:5212 为 Core 实例

## 背景
当访问 `localhost:5212/cecelia/console` 时，页面会重定向到 `/` 首页：
- `isCoreInstance()` 仅检查 hostname 中是否包含 "core"
- localhost:5212 不匹配，所以加载为 Autopilot
- Autopilot 没有 `/cecelia/console` 路由，导致重定向

## 成功标准
1. localhost:5212 被识别为 Core 实例
2. /cecelia/console 正常加载，不重定向
3. 支持 ?instance=core 查询参数用于测试
4. Core 专属路径（如 /cecelia, /ops）自动识别为 Core
