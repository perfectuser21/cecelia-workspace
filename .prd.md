# PRD: WorkerPanel 账户→模型→Skill 顺序重设计 (Frontend)

## 需求来源
用户要求 WorkerPanel 三个字段按"账户→模型→Skill"顺序排列，账户决定 provider，模型根据 provider 过滤。

## 功能描述

1. 字段顺序：Account → Model → Skill（移除独立 Provider 下拉）
2. Account 从 GET /api/brain/credentials 动态加载
3. 选账户后自动推断 provider，过滤模型列表
4. 每个账户 option 显示名称和文件路径

## 涉及文件
- apps/core/features/system/pages/TeamPage.tsx
- apps/core/features/system/api/staffApi.ts — 新增 fetchCredentials()
- apps/core/package.json — 版本 bump

## 成功标准
- 账户下拉从 API 动态拉取
- 选账户后模型列表按 provider 过滤
- tsc --noEmit 无报错

## 非目标
- 不修改 Brain API
- 不修改 cecelia-run
