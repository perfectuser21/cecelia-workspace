## PRD - 删除飞书登录，简化为纯本地访问

**需求来源**: 用户只通过 localhost 或 Tailscale (100.x.x.x) 访问系统，PR #257 已实现自动登录，飞书 OAuth 代码残留需清理。

**功能描述**: 移除所有飞书登录相关代码（FeishuLogin 页面、cookie 工具函数、登出弹窗、auth guard），简化 AuthContext 为始终以超级管理员身份登录。

**涉及文件**:
- 删除: `apps/dashboard/src/pages/FeishuLogin.tsx`
- 修改: `apps/dashboard/src/contexts/AuthContext.tsx` — 删除 cookie 逻辑，始终自动登录
- 修改: `apps/dashboard/src/App.tsx` — 删除 FeishuLogin import、登出弹窗、auth guard、/login 路由、登出按钮
- 修改: `apps/dashboard/src/vite-env.d.ts` — 删除飞书相关类型声明

**成功标准**:
1. `npx vite build` 编译通过
2. 浏览器 localhost:5211 直接进入，无登录页，无登出按钮
3. 侧边栏底部只显示超级管理员标识

**非目标**:
- 不改后端 `/api/feishu-login` 路由
- 不改 DynamicSidebar.tsx
