# PRD: Phase 4.2 - DLQ + Replay (失败不丢)

## 功能描述

实现死信队列(DLQ)机制，当 memory/evidence 写入失败时不丢失数据：
- memory 写失败 → memory_dlq.jsonl
- evidence 回传失败 → evidence_dlq.jsonl
- 提供 replay 接口和脚本

## 成功标准

- [ ] dlq/ 目录存储失败记录
- [ ] /api/system/dlq 查询 DLQ 状态
- [ ] /api/system/dlq/replay 重放失败记录
- [ ] scripts/replay-dlq.sh 一键重放
