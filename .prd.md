## PRD - 移除公网域名，改用本地地址

**需求来源**: 用户需求 - "core.zenjoymedia.media 替换成 perfect21:5211，dev-core.zenjoymedia.media 替换成 perfect21:5212"

**功能描述**:
Cecelia 系统不再通过公网域名访问，完全改为本地地址访问。需要在代码中替换所有域名引用。

**涉及文件**:
- `apps/dashboard/vite.config.ts` - 开发服务器 allowedHosts 配置
- `apps/dashboard/src/config/navigation.config.ts` - 注释中的域名引用
- `deploy/deploy-hk.sh` - 部署脚本中的输出信息
- `docs/LEARNINGS.md` - 文档中的域名引用（仅文档）

**成功标准**:
1. vite.config.ts 的 allowedHosts 中移除 `core.zenjoymedia.media` 和 `dev-core.zenjoymedia.media`
2. navigation.config.ts 中的注释改为本地地址
3. deploy-hk.sh 中的输出信息改为本地地址
4. 所有代码中不再出现 core.zenjoymedia.media / dev-core.zenjoymedia.media 的活跃引用
5. Cloudflare Tunnel 配置已从 version 28 更新到 version 29（已完成）

**非目标**:
- 不修改 autopilot 相关域名（zenjoymedia.media, dashboard.zenjoymedia.media 等）
- 不修改归档文档（.archive/ 目录下的文件）
- 不修改 Cloudflare Tunnel 配置（已在上一步完成）
