# PRD: 添加 Orchestrator Proxy

## 背景

cecelia-semantic-brain 已添加 `/orchestrator` API (端口 5220)，需要在 Core (5212) 加 proxy 让前端可以访问。

## 目标

在 Core 的 server.ts 中添加 proxy，将 `/api/orchestrator/*` 代理到 `http://localhost:5220/orchestrator/*`。

## 范围

- `apps/core/src/dashboard/server.ts` (添加 proxy)

## 验收标准

1. `curl http://localhost:5212/api/orchestrator/health` 返回 orchestrator 健康状态
2. 前端可以通过 `/api/orchestrator` 访问 orchestrator API
