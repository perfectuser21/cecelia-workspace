## PRD - Notion-like Task Database View

**需求来源**: 用户需要 Notion Database 风格的任务管理界面，替换当前只读的 OKR 树形视图
**功能描述**: 新建 TaskDatabase.tsx 组件，提供表格视图（Table）和看板视图（Board），支持列头排序、内联编辑 Priority/Status、拖拽改状态/重排序、多种分组方式（None/Status/Priority/Project/Month）
**涉及文件**:
- 新建: `apps/core/features/planning/pages/TaskDatabase.tsx`
- 修改: `apps/core/features/work/pages/WorkTabbed.tsx` (路由指向)
**成功标准**:
- `vite build` 通过
- `/work/tasks` 显示 Table 视图，列头可排序
- 切换 Board 视图，看板三列（Queued/In Progress/Completed），拖卡片改状态
- Group By 切换，表格按分组折叠显示
- 点击 Priority/Status 标签内联编辑正常
- 拖拽行重排序，刷新后顺序保持（localStorage）
**非目标**:
- 不改后端 API
- 不加新 npm 依赖
- 不加 Timeline 视图
- 拖拽排序不持久化到服务端
