# PRD: Phase 4.4 - Boundary Assertions (自动执法)

## 功能描述

实现数据边界断言，确保闭环数据的完整性：
- trace_id 必须存在且格式正确
- evidence 必须关联有效的 task_id
- memory payload 必须是有效 JSON

- [ ] 创建 assertions 模块

- [ ] evidence task_id 关联验证
- [ ] memory JSON 格式验证
