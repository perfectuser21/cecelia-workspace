# PRD: Phase 4.3 - Degrade Policy (猝死保护)

## 功能描述

实现优雅降级机制，当依赖服务不可用时：
- Brain 挂掉 → 降级到本地模式（写入 DLQ）
- Evidence 回传失败 → 降级到本地存储
- 服务恢复后 → 自动重连并重放 DLQ

## 成功标准

- [ ] 健康检查机制 (定期探测依赖服务)
- [ ] 降级状态管理 (degraded mode flag)
- [ ] 自动恢复检测 (服务恢复后重连)
- [ ] DLQ 集成 (降级时写入 DLQ)
