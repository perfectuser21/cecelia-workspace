# PRD: TeamPage 通用 Credentials 支持

## 背景

TeamPage.tsx 中账户选择（credentialsFile）只针对 MiniMax provider 显示，其他 provider 无法选择账户。

## 目标

把 MINIMAX_ACCOUNTS 改为通用的 PROVIDER_ACCOUNTS，对所有 provider 显示账户选择器。

## 需求

1. 将 `MINIMAX_ACCOUNTS` 常量改为 `PROVIDER_ACCOUNTS: Record<string, string[]>`
2. WorkerPanel 初始 credentialsFile state 默认使用 `credentials_file || provider || 'anthropic'`
3. save 函数中移除 `provider === 'minimax' ?` 条件，对所有 provider 传 credentials_file
4. provider select onChange 时同步更新 credentialsFile 为新 provider 的默认值
5. 账户下拉对所有 provider 显示（移除 `provider === 'minimax' &&` 条件）

## 验收标准

- tsc --noEmit 无报错
- 所有 provider 都显示账户选择器
- 切换 provider 时账户默认值自动更新
