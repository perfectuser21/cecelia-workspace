## PRD - TaskDatabase 丝滑度优化（P0）

**需求来源**: 用户反馈当前 TaskDatabase 交互不够丝滑，需要优化手感
**功能描述**: 应用 Notion 的 UX 最佳实践，提升 TaskDatabase 的交互体验

**P0 优化清单**：
1. **拖拽手柄 hover 显示**：默认隐藏，鼠标悬停行时才显示（`opacity-0 group-hover:opacity-100`）
2. **拖拽时 scale 放大**：拖起时 `scale(1.02)` + 增强阴影，视觉反馈更明显
3. **行高增加**：`py-2.5` → `py-3`（32px → 36px），减少拥挤感
4. **下拉菜单动画**：Priority/Status 下拉菜单加入淡入 + 上滑动画（200ms transition）

**涉及文件**:
- 修改: `apps/core/features/planning/pages/TaskDatabase.tsx`

**成功标准**:
- vite build 通过
- 拖拽手柄默认隐藏，hover 行时显示
- 拖拽卡片时有 scale 放大效果
- 表格行高视觉上更舒适
- 下拉菜单打开/关闭有平滑动画

**非目标**:
- 不做骨架屏（P1）
- 不做毛玻璃效果（P2）
- 不做 Table drop zone 高亮（P1）
