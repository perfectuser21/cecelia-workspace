# PRD: 修复 CORS 缺少 PUT 方法

## 功能描述

修复 `apps/core/src/dashboard/server.ts` CORS 配置缺少 `PUT` 方法，导致模型配置切换（Anthropic 主力预设）按钮点确定后一直转圈的问题。

根因：`Access-Control-Allow-Methods` 未包含 `PUT`，浏览器 OPTIONS preflight 被拒绝，实际 PUT 请求永远不发出。

## 成功标准

- 点击"Anthropic 主力"→ 确定，切换成功，不再转圈
- `PUT /api/brain/model-profiles/active` 通过 5211 正常响应

## 修改范围

`apps/core/src/dashboard/server.ts` 第 60 行，加入 `PUT`。
