## PRD - 品牌重塑：悦升云端 → Perfect21

**需求来源**: 系统是个人管家工具，不再使用公司品牌"悦升云端"，改为个人品牌"Perfect21"。

**功能描述**: 将所有品牌标识从"悦升云端"替换为"Perfect21"，包括 Core 实例配置、HTML 标题、meta 描述、侧边栏 fallback 值。收起时 logo 字母显示 P。

**涉及文件**:
- `apps/core/features/config.ts` — Core 实例 name 和 logoCollapsed
- `apps/dashboard/index.html` — HTML title 和 meta description
- `apps/dashboard/src/App.tsx` — fallback 值（'Z' → 'P', '运营中台' → 'Perfect21'）
- `apps/dashboard/src/components/DynamicSidebar.tsx` — 同上 fallback 值

**成功标准**:
1. `npx vite build` 编译通过
2. 浏览器 localhost:5211 侧边栏 logo 收起显示 P
3. 页面标题显示 Perfect21
4. 无任何"悦升云端"残留

**非目标**:
- 不改 logo 图片文件（仍用 logo-white.png）
- 不改 PWA manifest（已经是 Cecelia）
- 不改后端
