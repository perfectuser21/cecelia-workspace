# PRD - Brain Status Dashboard

**需求来源**: 用户需要 Brain 状态可视化 Web Dashboard

**功能描述**:
实现一个实时 Web Dashboard，可视化展示 Cecelia Brain 的运行状态，包括：
- 核心状态面板：健康状态、Tick 信息、三层大脑（L0 脑干/L1 丘脑/L2 皮层）、保护系统（警觉/熔断器/隔离区/看门狗）
- OKR 进度面板：当前聚焦的 Objective 和 KR 列表，进度可视化
- 任务队列面板：queued/in_progress/completed 任务统计和详情

**涉及文件**:
- `apps/core/features/execution/pages/BrainStatusDashboard.tsx` - 主 Dashboard 页面
- `apps/core/features/execution/components/BrainCoreStatus.tsx` - 核心状态组件
- `apps/core/features/execution/components/BrainOKRProgress.tsx` - OKR 进度组件
- `apps/core/features/execution/components/BrainTaskQueue.tsx` - 任务队列组件
- `apps/core/features/execution/api/brain.api.ts` - Brain API 调用函数
- `apps/core/features/execution/index.ts` - 路由注册
- Brain API 端点已存在，无需修改后端

**成功标准**:
- ✅ 访问 `http://perfect21:5212/brain-status` 显示完整 Dashboard
- ✅ 自动刷新（5 秒间隔）
- ✅ 显示核心状态、OKR 进度、任务队列三个面板
- ✅ 数据来自 Brain API (`/api/brain/*`)
- ✅ 响应式设计，支持移动端

**非目标**:
- ❌ 不实现 CLI 工具
- ❌ 不实现告警推送（P1 功能）
- ❌ 不实现历史趋势分析（P2 功能）
- ❌ 暂不实现学习记录面板和资源监控面板（P1 功能）

**技术方案**:
- 使用现有 React + TypeScript 架构
- 集成到 execution feature module
- 使用 shared/hooks/useApi.ts（SWR 模式）实现数据缓存和自动刷新
- 使用 shared/components 现有组件库（Card/Badge/Progress 等）
