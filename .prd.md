# PRD: CeceliaPage 重设计 - Conversational Briefing

## 功能描述

重写 CeceliaPage.tsx，从线性堆叠改为三区布局的对话式简报：红区（需要决策 desires），黄区（待确认任务），聊天区（默认折叠历史）。按钮直接调真实 API，取消 opening LLM 调用。

## 成功标准

- 首屏 30 秒内看到简报+决策+待确认任务
- [派发] 按钮调用 POST /api/brain/tasks/:id/dispatch
- [了解] 按钮调用 PATCH /api/brain/desires/:id 标记 acknowledged
- Desire 去重分组，不显示重复内容
- 聊天默认 3 条，不自动滚底
- npm run build 通过
