# PRD - Intent 识别模块增强

## 需求来源

TRD Milestone 1: "理解自然语言意图并转化为结构化任务"

当前 `apps/core/src/brain/intent.js` 已实现基础功能，但存在局限性需要增强。

## 功能描述

增强 Intent 识别模块的短语模式匹配和实体提取能力：

1. **扩展短语模式**：增加更多自然语言表达的匹配模式
2. **增强实体提取**：支持更多实体类型（优先级、时间范围、依赖关系）
3. **改进置信度计算**：区分高/中/低置信度，提供更准确的分类结果

## 涉及文件

- apps/core/src/brain/intent.js（修改）
- apps/core/src/brain/__tests__/intent.test.js（修改）

## 成功标准

- [ ] 扩展 INTENT_PHRASES 增加至少 5 个新短语模式
- [ ] ENTITY_PATTERNS 增加 priority、timeframe、dependency 实体类型
- [ ] 置信度计算区分高(>=0.7)/中(0.4-0.7)/低(<0.4) 三档
- [ ] 所有现有测试通过
- [ ] 新功能有对应测试覆盖
