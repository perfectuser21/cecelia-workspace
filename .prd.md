## PRD - Intent-to-Action 映射增强

**需求来源**: Milestone 1: 意图识别系统 (KR1) - 第3个验收项

**功能描述**:
增强 `intent/execute` 端点，使其通过 `handleAction` 执行 brain action（白名单校验、幂等性、决策日志），补全 `queryStatus` handler 查询任务/目标状态，增加置信度阈值检查防止低置信度自动执行。

**涉及文件**:
- `apps/core/src/brain/routes.js` — 修改 intent/execute 路由 + 添加 executeQueryStatus 函数
- `apps/core/src/brain/__tests__/intent.test.js` — 新增测试

**成功标准**:
- [x] intent/execute 通过 handleAction 执行 action 意图（替代 inline ACTION_HANDLERS）
- [x] queryStatus handler 返回任务/目标状态数据
- [x] 置信度阈值参数可配置，低置信度时不执行
- [x] dry_run 模式返回解析结果但不执行
- [x] 决策日志自动记录
- [x] 99 个测试通过

**非目标**:
- 不修改 intent.js 核心解析逻辑
- 不修改前端
