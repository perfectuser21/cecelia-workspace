# Agent 模型配置 UI — 每个 Agent 单独选模型

## 功能描述

升级模型 Profile 页面，支持每个 LLM Agent 单独选择模型，取代原来的整体 profile 切换。包括预设快速切换条、Agent 配置表格、乐观更新和 WebSocket 实时同步。

## 涉及文件

- apps/core/features/system/api/model-profile.api.ts
- apps/core/features/system/components/ProfilePresetBar.tsx (新建)
- apps/core/features/system/components/AgentModelTable.tsx (新建)
- apps/core/features/system/pages/ModelProfileSwitcher.tsx

## 成功标准

- 页面分为预设条 + Agent 配置表格
- 选择模型后即时 PATCH 生效
- 固定 provider 的 agent 显示锁定标识
- 预设切换重置所有配置
- WebSocket 实时更新

## 非目标

- 不修改后端 Brain API
- 不修改路由注册
