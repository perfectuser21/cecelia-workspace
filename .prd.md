# PRD: Orchestrator Proxy + 语音 UI

## 背景

Orchestrator API 已整合到 Brain (cecelia-semantic-brain)，包括：
- `/orchestrator/chat` - 聊天
- `/orchestrator/voice/tts` - 文字转语音
- `/orchestrator/voice/stt` - 语音转文字

Workspace 需要：
1. 移除本地 orchestrator routes
2. 代理所有 orchestrator 请求到 Brain
3. 前端添加语音 UI

## 目标

1. 删除本地 `src/orchestrator/routes.ts`
2. 更新 `server.ts` 代理配置
3. 前端添加麦克风按钮和语音播放

## 架构

```
Browser → Workspace (5211/5212) → Proxy → Brain (5220)
                                    ↓
                            /api/orchestrator/*
```

## 验收标准

1. 本地 orchestrator routes 已删除
2. 所有 `/api/orchestrator/*` 请求代理到 Brain
3. 前端可以语音输入
4. 前端可以播放语音回复
