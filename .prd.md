# PRD: 修复 CeceliaChat 响应格式解析

## 功能描述
CeceliaChat 前端组件发消息后没有反应，因为响应解析格式与 Brain API 实际返回不匹配。

## 问题分析
Brain API 返回 `{ reply, routing_level, intent }`，但前端期望 `{ success: true, response: { message } }`。
`data.success` 为 undefined，条件永远为 false，回复永远不显示。

## 成功标准
1. CeceliaChat 发消息后能正确显示 Brain API 回复
