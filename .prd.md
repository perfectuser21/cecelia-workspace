## PRD - 导航架构重构：从平层到层级深钻

**需求来源**: 用户反馈当前网站导航"很大的平层"，与飞书/Notion/Linear 等主流平台相比显得业余

**功能描述**: 让侧边栏支持展开/折叠子项，暴露当前隐藏在 tabs 里的页面，并添加面包屑导航显示当前位置。具体包括：
1. 类型系统扩展 - NavGroupItem/FeatureRoute.navItem/NavItem 添加 children 字段
2. Feature Manifests 更新 - today/work/system-hub/knowledge 声明子导航项
3. buildNavGroupsFromManifests() 传递 children
4. 新建 CollapsibleNavItem 组件 - 可展开/折叠的侧边栏项
5. 更新 3 处 convertCoreNavGroups() 递归处理 children
6. 更新侧边栏渲染 - 有 children 用 CollapsibleNavItem
7. 新建 Breadcrumb 组件 - 面包屑导航
8. filterNavGroups() 递归过滤 children

**涉及文件**:
- `apps/core/features/types.ts` - 修改
- `apps/dashboard/src/config/navigation.config.ts` - 修改
- `apps/core/features/index.ts` - 修改
- `apps/core/features/today/index.ts` - 修改
- `apps/core/features/work/index.ts` - 修改
- `apps/core/features/system-hub/index.ts` - 修改
- `apps/core/features/knowledge/index.ts` - 修改
- `apps/dashboard/src/components/CollapsibleNavItem.tsx` - 新建
- `apps/dashboard/src/components/Breadcrumb.tsx` - 新建
- `apps/dashboard/src/App.tsx` - 修改
- `apps/dashboard/src/components/DynamicRouter.tsx` - 修改
- `apps/dashboard/src/components/DynamicSidebar.tsx` - 修改

**成功标准**:
1. 前端 build (`cd apps/dashboard && npx vite build`) 无 TypeScript 错误
2. 后端 build (`cd apps/core && npx tsc --noEmit`) 编译通过
3. 侧边栏 Today/Work/Knowledge/System 显示 chevron，可展开/折叠子项
4. 子项可点击导航到对应页面，当前页面子项高亮
5. 面包屑显示 `System > Automation` 格式
6. Dashboard 保持 flat（无子项）
7. 侧边栏收缩时只显示父级 icon
8. Autopilot 实例不受影响
