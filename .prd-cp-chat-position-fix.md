# 聊天位置和导航优化

## 功能描述

修复 Cecelia 聊天组件位置，并优化导航体验。

## 成功标准

1. 聊天按钮和面板都在右下角
2. 聊天面板固定宽高 (w-80 h-[480px])
3. 导航时自动收起侧边栏
4. 导航提示更大更明显

## 实现方案

### 1. 聊天位置
- 按钮: fixed bottom-6 right-6
- 面板: fixed bottom-20 right-6 (按钮上方)
- 固定尺寸: w-80 h-[480px]

### 2. 导航时收起侧边栏
- CeceliaProvider 添加 onNavigate 回调
- App.tsx 传入收起侧边栏的函数
- navigate_to 执行时调用

### 3. 更明显的导航提示
- 增大尺寸和发光效果
- 更长的显示时间
