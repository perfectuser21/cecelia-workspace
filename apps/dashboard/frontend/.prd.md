# PRD - 健康检查历史记录增强

## 背景

当前 ServiceHealthCard 组件只显示最新一次的健康检查结果。增加历史记录展示可以让用户更好地了解服务的稳定性趋势。

## 功能需求

### 1. 前端模拟历史数据
- 在前端生成模拟的健康检查历史记录
- 每次刷新时追加新的检查记录
- 保留最近 10 条历史记录

### 2. 历史记录展示
- 在 ServiceHealthCard 展开时显示历史记录
- 显示时间戳、状态、延迟信息
- 健康/异常状态有明显视觉区分（绿色/红色）

## 成功标准

- [ ] 展开卡片时显示健康检查历史记录
- [ ] 历史记录包含时间戳、状态、延迟
- [ ] 状态有颜色区分
- [ ] npm run build 通过
