# PRD - 健康检查增强

## 背景

当前 PerformanceMonitoring 页面已有服务健康状态显示，但功能较简单。需要增强以提供更好的用户体验。

## 目标

增强前端健康检查显示，让用户能看到更多有用信息，并提供更好的交互体验。

## 功能需求

### 1. 服务卡片增强
- 添加展开/收起功能，点击可查看详情
- 展开后显示完整检查时间、服务标识等
- 延迟显示带颜色标识（优秀/正常/较慢）

### 2. 手动刷新功能
- 每个服务卡片添加刷新按钮
- 刷新时显示加载动画
- 触发整体健康检查刷新

### 3. 提取独立组件
- 将服务健康卡片提取为 ServiceHealthCard 组件
- 提高代码复用性和可维护性

## 非功能需求

- 复用现有 systemApi.getHealth() API
- 保持 UI 简洁
- 支持 Dark Mode

## 成功标准

- [ ] 服务卡片可展开查看详情
- [ ] 支持手动刷新（带加载动画）
- [ ] 延迟显示带颜色标识
- [ ] 代码通过 lint 和类型检查
- [ ] 构建成功
