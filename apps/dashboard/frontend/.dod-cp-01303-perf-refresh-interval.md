# DoD - 性能监控刷新间隔设置

QA: docs/QA-DECISION.md

## 验收标准

### 功能验收
- [ ] 用户可从预设选项选择刷新间隔（10s / 30s / 60s / 暂停）
      Test: auto:TypeScript 类型检查
- [ ] 选择后立即生效，定时器按新间隔运行
      Test: manual:目视验证选择后刷新行为变化
- [ ] 状态在页面会话中保持
      Test: manual:切换页面后返回，验证设置保留
- [ ] UI 清晰显示当前刷新间隔设置
      Test: manual:目视验证下拉框或按钮组显示当前值

      Test: auto:TypeScript 编译
