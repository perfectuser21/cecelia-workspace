# Changelog

All notable changes to @cecelia/autopilot-frontend will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-01-31

### Added

- Health rate (uptime percentage) display in ServiceHealthCard component
- Color-coded health rate indicator (green >= 99%, amber >= 95%, red < 95%)

## [1.0.0] - 2026-01-27

### Added

- Initial release of @cecelia/autopilot-frontend
- Dashboard interface for social media automation
- Feature Dashboard page for managing development features
- Real-time task visualization
- GitHub integration UI (branches, PRs, commits)
- Dev workflow tracking UI
- Responsive design with Tailwind CSS
- Dark mode support

### Changed

- Migrated from standalone social-metrics-frontend to monorepo
- Updated package name from `social-metrics-frontend` to `@cecelia/autopilot-frontend`
- Updated TypeScript path aliases to point to monorepo structure
