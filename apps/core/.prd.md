## PRD - Cecelia 静默层 Phase 3：汇报系统 + Planner V2

**需求来源**: Cecelia 自我复活计划（docs/CECELIA-REVIVAL-PRD.md Phase 3）

**功能描述**: 为 Cecelia 添加汇报能力和自主规划能力：
1. Notifier — 飞书 webhook 推送：任务完成/失败/熔断/日报
2. Planner V2 — 队列空时根据 KR progress 和 gap 自动生成下一个任务

**涉及文件**:
- 新建 `src/brain/notifier.js` — 飞书推送模块
- 修改 `src/brain/planner.js` — 升级 generateNextTask 为 V2
- 修改 `src/brain/routes.js` — 集成 Notifier 到 execution-callback
- 修改 `src/brain/tick.js` — 集成 Notifier 到 patrol/circuit-breaker 事件

**成功标准**:
1. 任务完成后飞书群收到 ✅ 通知
2. 任务失败后飞书群收到 ❌ 通知
3. 熔断触发后飞书群收到 ⚠️ 通知
4. Planner V2：队列空时自动生成新任务写入 DB
5. brain/ 目录下所有测试通过（不退化）

**非目标**:
- 不实现每日 22:00 日报（Phase 4 自驱时再加 cron）
- 不动业务代码
- 不改前端
