# DoD: Cecelia Voice + Orchestrator 队列管理集成

## 功能验收

### 1. Orchestrator 队列管理 API
- [ ] `GET /api/orchestrator/queue` 端点实现
  - [ ] 返回 queued 任务列表
  - [ ] 返回 running 任务列表
  - [ ] 返回统计信息（queued_count, running_count, available_slots）
  
- [ ] `POST /api/orchestrator/execute-now/:id` 端点实现
  - [ ] 可以找到队列中的任务
  - [ ] 如果有空闲槽位，立即执行
  - [ ] 如果没有空位，插队到第一位
  
- [ ] `POST /api/orchestrator/pause/:id` 端点实现
  - [ ] 可以找到运行中的任务
  - [ ] 从运行队列移除
  - [ ] 返回释放的槽位信息

### 2. Realtime Voice Tools
- [ ] Realtime Config 包含 3 个工具定义
  - [ ] `get_queue` - 查看队列状态
  - [ ] `execute_now` - 插队执行（支持模糊匹配）
  - [ ] `pause_task` - 暂停任务（支持模糊匹配）

- [ ] 工具执行逻辑实现
  - [ ] `get_queue` 返回格式化的队列摘要
  - [ ] `execute_now` 调用队列管理 API
  - [ ] `pause_task` 调用队列管理 API
  - [ ] 所有工具返回友好的文本反馈

### 3. 集成测试
- [ ] 可以通过语音调用 `get_queue` 查看队列
- [ ] 可以通过语音调用 `execute_now` 插队任务
- [ ] 可以通过语音调用 `pause_task` 暂停任务
- [ ] 错误情况有正确的提示（任务不存在等）
