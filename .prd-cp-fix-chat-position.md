# 修复聊天位置

## 功能描述

修复 Cecelia 聊天窗口位置问题。Tailwind 没有生成 `bottom-6` 类，导致聊天窗口位置错误显示在左上角。

## 成功标准

1. 聊天按钮显示在右下角 (bottom: 24px, right: 24px)
2. 聊天面板显示在右下角，固定 480px 高度
3. 使用 inline style 确保位置正确渲染
